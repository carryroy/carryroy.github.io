Android Studio快捷操作

1.File - close project 关闭整个工作组 回到最开始界面，按delete删除recent projects

2.从Eclipse中导入项目：右键项目名，export -- Android -- general gradle...

3.appearance设置主题

4.新建App-----右键projects-----选择new Module-----选择EmptyActivity

5.setting + edit + general 里的other 里的选项 show quick doc on mouse move 设置鼠标悬浮显示描述，但是很消耗性能

6.setting + edit + general + code completion 里的case sensitive completion是大小写敏感，选择none

快捷键 作用
Ctrl + tab 在不同的选项卡中切换
alt + enter 提示错误
Ctrl + shift + enter 系统自动补全
Ctrl + alt + O 管理引用
Ctrl + Alt +Space 提示输入
Ctrl + Q 放在关键字上看文档描述
Ctrl + P 查看方法的参数
Ctrl + shift + space suggestion例如新建一个animation提示哪种类型
在一句代码后Ctrl + Alt + T 包围语句，比如if
Ctrl + J 出现如for循环等语句
Ctrl + shift + J 选中多行代码合为一行
Ctrl + Alt + L 格式化代码
Alt + Insert 生成代码 如get set toString
Ctrl + alt + V 创建局部变量，相当于eclipse中Ctrl+2+L
Ctrl + alt + F 把局部变量变为成员或构造
Ctrl + H 把鼠标移到一个类上看继承关系
Ctrl + N 查找类
.cast 类型转换
.field 提取成员变量
.var 提取局部变量
